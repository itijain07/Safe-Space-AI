# 🛡️ Safe Space AI

Safe Space AI is an AI-powered journaling web application that helps users reflect on their thoughts through mood analysis and personalized insights.

## Features

- Write journal entries
- AI mood analysis
- Journal history
- Insights dashboard
- Reflection questions
- Dark mode UI

## Tech Stack

- React
- TypeScript
- Vite
- Express.js
- Tailwind CSS
